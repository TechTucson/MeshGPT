## About
MeshGPTUltimate is an AI Meshtastic Chatbot running locally on edge computing hardware.

## How to Install 

### Installing  (Ubuntu Specific)
- ```mkdir projects ```
- ```cd projects```
  - git clone https://github.com/CadenMW/MeshGPT.git
  -cd MeshGPT
  - ``` python -m venv venv```
  - ```sourcce venv/bin/activate```
  - ``` pip install -r requirements.txt ```
### Running the Server
- ``` python3 meshgptultimate.py ```
  - To stop the server use CTRL-C.
